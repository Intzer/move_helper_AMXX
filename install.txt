Информация по установке:
1. Раскидать файлы по папкам как они лежат
2. Узнаём свою версию AMXX: в консоль сервера на хостинге пишем amxx version, консоль нам выведет версию
3. Для версии AMXX меньше 1.9.0 выбрать плагины из папки addons/amxmodx/plugins/AMXX_182, для версии AMXX 1.9.0 и выше - из addons/amxmodx/plugins
